# Spinner Icon

A lightweight animated spinner that indicates a loading or in-progress state.
It inherits its color from `currentColor` and can be sized via the `size` input
or by letting it scale with the surrounding text (default `1em`).

## Usage

Add `DmaSpinnerIconComponent` to your component's `imports` array:

```ts
import { Component, ChangeDetectionStrategy } from '@angular/core';
import { DmaSpinnerIconComponent } from '@dnd-mapp/arcane-ui/icons';

@Component({
    selector: 'app-example',
    templateUrl: `./example.html`,
    changeDetection: ChangeDetectionStrategy.OnPush,
    imports: [DmaSpinnerIconComponent],
})
export class ExampleComponent {}
```

And then in the component's template

```html
<!-- default (inherits font size) -->
<dma-icon-spinner />

<!-- explicit size -->
<dma-icon-spinner size="lg" />
```
