# Icons

A set of lightweight SVG icon components. Each icon inherits its color from
`currentColor`, is `aria-hidden` by default, and can be sized via the `size`
input or by letting it scale with the surrounding text (default `1em`).

## Usage

Add the icon component(s) you need to your component's `imports` array:

```ts
import { SpinnerIconComponent } from '@dnd-mapp/arcane-ui/icons';
```

Then use the selector in your template:

```html
<!-- scales with surrounding text (default) -->
<dma-icon-spinner />

<!-- explicit size -->
<dma-icon-spinner size="lg" />
```

All icons accept the same `size` input (`xs` | `sm` | `md` | `lg` | `xl`). Omit
it to let the icon scale with the surrounding font size (`1em`).

Because icons are `aria-hidden`, always pair them with an accessible label on
the parent element:

```html
<button aria-label="Delete item">
    <dma-icon-trash />
</button>
```
