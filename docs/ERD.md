```mermaid
    users {
        string id       PK

    }
    
    authorization_transactions {
        string id               PK
        string code_challenge
        string login_challenge
        string state

        string authorization_code           "N"
        string authorization_code_expiry    "N"
    }

        string      id              PK
        string      redirect_urls

    }
    
    clients ||--o{ authorization_transactions : Executes
```

## Authorization Transaction

The `nonce` is used for the `nonce` claim of the Identity token. This value is generated by the client that initializes the Authorization flow.
